# 홈페이지 스토리북에 대하여

홈페이지에 대한 스토리북입니다. 각 페이지 컴포넌트에 사용되는 mdx 파일의 관리를 어떻게 해야하는지 설명해 두었습니다.

# mdx파일 보는법

### 파일 디렉토리

해당 파일의 디렉토리를 나타냅니다. 나중에 베타서버까지 배포된 후에는 해당 파일에 깃헙 링크를 걸어두도록 하겠습니다.

```
ex)
`/content/recruit/jobPosition.mdx`
```

### 예시

해당 mdx파일이 어떻게 사용되고 있는지 예시코드가 보여집니다.

```
ex)
---
title: 마스터즈 코스
subtitle: 최선과 최고의 선택
---
```

### 항목별 설명

위에서 예시로 나온 항목들을 간단하게 설명해둔 항목입니다.

```
ex1)

* title: 제목을 입력할 수 있습니다.
* subtitle: 소제목에 대해 입력할 수 있습니다.
```

위의 예시 문단에서 항목들이 보여질 때 아래와 같은 형식이라면 해당 항목은 그와 같은 형태로 입력되어야 합니다. '입력할 수 있습니다.' 문구가 붙은 값들만 해당 값에 대해 직접적으로 입력하는 것이며,
아닌 경우에는 여러가지 다른 방법으로 입력하게 됩니다. 이런경우에 인덴트를 확인하거나 예시를 확인하면 쉽게 파악할 수 있습니다.

```
ex2)
career:

- 초등학교 졸업
- 중학교 졸업
- 대학교 졸업

ex2) 해당 경우에는 모든 컬럼에 붙던
interviewList

- title: 최고의 선택이었다.
  description: 동해물과 백두산이 마르고 닳도록 하느님이 보우하사 우리나라만세
  writer: micky

- title: 최악의 선택이었다.
  description: 이몸이 죽고죽어 일백번 고쳐죽어
  writer: Skeleton

```
